# 数据源研究结果

## 1. GLOBOCAN 数据库

### 基本信息
- **管理机构**: IARC (International Agency for Research on Cancer)
- **网站**: https://gco.iarc.fr/
- **最新版本**: GLOBOCAN 2022
- **覆盖范围**: 185个国家/地区，36种癌症类型

### 数据访问方式
**Web界面访问**：通过Cancer Today工具（https://gco.iarc.who.int/today/en）提供交互式数据可视化，包括：
- 条形图、饼图、热力图、散点图、表格等多种可视化方式
- 按国家、癌症类型、人口分组查看数据
- 可下载高分辨率图表

**数据下载限制**：
- 没有公开的API接口
- 主要通过Web界面进行数据查询和可视化
- 可以下载图表和fact sheets（PDF格式）
- 无法批量下载原始数据

**数据类型**：
- 发病率（Incidence）：新发病例数、年龄标准化发病率（ASR）
- 死亡率（Mortality）：死亡病例数、年龄标准化死亡率
- 患病率（Prevalence）：1年、3年、5年患病率

**数据分层**：
- 地理分层：全球、区域、次区域、国家
- 人口分层：性别、年龄组
- 社会经济分层：人类发展指数（HDI）、收入水平、WHO区域、UN区域

### 数据获取策略
由于GLOBOCAN没有公开API，需要采用以下策略：
1. **Web爬虫**：使用浏览器自动化工具抓取Cancer Today的数据表格
2. **手动下载**：下载关键的fact sheets和预生成的数据表
3. **数据模拟**：基于已发表的论文数据和公开的fact sheets创建示例数据集
4. **引导用户**：提供数据上传功能，让用户手动从GLOBOCAN下载数据后上传

## 2. GBD (Global Burden of Disease) 数据库

### 基本信息
- **管理机构**: IHME (Institute for Health Metrics and Evaluation)
- **网站**: https://vizhub.healthdata.org/gbd-results/
- **最新版本**: GBD 2023
- **时间跨度**: 1990-2023年

### 数据访问方式
**GBD Results Tool**：
- **网址**: https://vizhub.healthdata.org/gbd-results/
- **访问要求**: 需要注册和登录
- **数据查询**: 可自定义查询条件
  - 测量指标（Measure）：Deaths, DALYs, YLLs, YLDs等
  - 指标类型（Metric）：Number, Percent, Rate
  - 病因（Cause）：292种疾病和伤害原因
  - 地理位置（Location）：全球、区域、国家
  - 年龄组（Age）：各年龄段
  - 性别（Sex）：男性、女性、两性
  - 年份（Year）：1990-2023

**数据下载**：
- **格式**: CSV文件
- **下载方式**: 通过Web界面查询后下载
- **访问限制**: 需要注册账户并同意使用条款

**API接口**：
- **无公开API**: GBD没有提供公开的REST API
- **数据访问**: 必须通过Web界面进行

### 风险因素数据
GBD提供以下风险因素的流行率数据：
- 感染因素：HBV、HCV等
- 代谢因素：肥胖、糖尿病、MASLD等
- 行为因素：吸烟、饮酒等
- 环境因素：空气污染、黄曲霉毒素等

### 数据获取策略
1. **用户注册引导**：引导用户注册IHME账户
2. **数据查询模板**：提供预设的查询参数模板
3. **CSV上传**：允许用户从GBD Results Tool下载CSV后上传
4. **示例数据**：提供基于GBD 2021的示例数据集用于演示

## 3. CI5 (Cancer Incidence in Five Continents) 数据库

### 基本信息
- **管理机构**: IARC
- **网站**: https://ci5.iarc.fr/
- **最新版本**: CI5plus (更新至2024年10月)
- **覆盖范围**: 125个癌症登记处人口，最长时间跨度至2017年

### 数据访问方式
**数据下载**：
- **下载页面**: https://ci5.iarc.fr/ci5plus/download
- **数据格式**: CSV (Comma Separated Value)
- **访问限制**: 免费使用，但不得用于商业或促销目的

**两种数据库**：
1. **CI5plus Summary**（汇总版）：
   - 包含年度汇总数据（无个人记录）
   - 用于在线分析工具的数据
   
2. **CI5plus Detailed**（详细版）：
   - 包含125个登记处人口的更新年度发病率
   - 最长时间跨度至2017年
   - 覆盖所有癌症和28种主要类型
   - 可用于时间趋势分析

**数据内容**：
- 癌症发病率数据
- 按性别、5岁年龄组、癌症类型分类
- 组织学亚型数据

### 数据获取策略
1. **直接下载**：CI5数据可以直接下载CSV文件
2. **数据解析**：解析CSV文件并导入数据库
3. **数据整合**：与GLOBOCAN和GBD数据整合

## 综合数据获取策略

### 方案A：完全自动化（技术难度高）
1. **GLOBOCAN**: 使用浏览器自动化（Selenium/Playwright）爬取数据
2. **GBD**: 自动化登录和数据查询（需要用户提供账户信息）
3. **CI5**: 直接下载和解析CSV文件

**优点**：用户体验最佳，完全自动化
**缺点**：技术复杂，可能违反服务条款，维护成本高

### 方案B：半自动化（推荐）
1. **数据上传功能**：
   - 用户从GLOBOCAN、GBD手动下载数据
   - 上传CSV/Excel文件到平台
   - 系统自动解析和整合数据

2. **CI5自动下载**：
   - CI5数据可以直接下载，系统自动获取

3. **示例数据集**：
   - 提供基于参考论文的示例数据集
   - 用户可以直接使用示例数据进行演示

**优点**：
- 符合数据使用条款
- 技术实现简单
- 用户可控性强

**缺点**：
- 需要用户手动下载部分数据
- 不是完全自动化

### 方案C：示例数据 + 模板（MVP版本）
1. **预置示例数据**：
   - 基于参考论文的HCC数据
   - 覆盖主要国家和风险因素

2. **研究方案模板**：
   - 提供标准的研究方案模板
   - 用户填写研究参数

3. **数据上传选项**：
   - 允许用户上传自己的数据
   - 支持CSV、Excel格式

**优点**：
- 快速实现MVP
- 展示完整功能流程
- 降低技术风险

**缺点**：
- 数据不是实时的
- 覆盖范围有限

## 推荐实施方案

**第一阶段（MVP）**：采用方案C
- 使用示例数据集（基于参考论文的HCC数据）
- 实现完整的分析和论文生成流程
- 提供数据上传功能

**第二阶段（增强）**：升级到方案B
- 实现CI5数据的自动下载
- 完善数据上传和解析功能
- 提供数据下载指南

**第三阶段（完整）**：部分采用方案A
- 对于允许爬取的数据源，实现自动化
- 保持用户上传选项作为备选方案

## 数据格式标准化

### 统一数据模型
所有三个数据源的数据需要标准化为统一格式：

```
{
  "country": "国家/地区",
  "region": "区域",
  "year": "年份",
  "cancer_type": "癌症类型",
  "age_group": "年龄组",
  "sex": "性别",
  "incidence_count": "发病例数",
  "incidence_rate": "发病率",
  "mortality_count": "死亡例数",
  "mortality_rate": "死亡率",
  "prevalence": "患病率",
  "risk_factors": {
    "HBV": "乙肝病毒PAF",
    "HCV": "丙肝病毒PAF",
    "obesity": "肥胖PAF",
    "diabetes": "糖尿病PAF",
    "alcohol": "酒精PAF",
    "smoking": "吸烟PAF",
    ...
  }
}
```
